This project aims to classify websites as phishing or legitimate using a dataset from the UCI Machine Learning Repository. Phishing websites attempt to trick users into revealing sensitive information, and this model helps in identifying such malicious sites based on their attributes.

  • Features: 30
  • Target Variable: result

ü§ñ Model Training Overview

We trained multiple classification algorithms to predict whether a website is phishing or legitimate. The models included:

  • Logistic Regression
  • Decision Tree Classifier
  • Random Forest Classifier
  • Gradient Boosting Classifier
  • AdaBoost Classifier
  • K-Nearest Neighbors (KNN)

Each model was tuned using hyperparameter grids to optimize performance.

The best-performing model was selected based on cross-validation accuracy and generalization on the test set.
